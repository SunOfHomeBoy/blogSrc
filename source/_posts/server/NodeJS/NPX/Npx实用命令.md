# Npx简介
`Npx`可以查找本地`node_modules`包下的 库文件，所有 工具包 不需要 全局安装即可通过`Npx`找到。 

## 实用命令
```bash
$> npx nrm use taobao       # 无须安装 nrm 即可使用
$> npx http-server -p 3000  # 快速启动 静态文件服务器 -p 指定端口号
```
